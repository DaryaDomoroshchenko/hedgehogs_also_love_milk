# [Hedgehogs also love milk](https://DaryaDomoroshchenko.github.io/hedgehogs_also_love_milk/)
## Ёжики тоже любят молоко
### Используемый стек технологий: HTML, CSS, JS, jQuery, Git, Webpack
Адаптивный лендинг для любителей ёжиков.
***
### Для запуска проекта необходимо:
1. Установить [Nodejs](https://nodejs.org/en/)
2. `git clone git@github.com:DaryaDomoroshchenko/hedgehogs_also_love_milk.git` - клонировать репозиторий
3. `npm i` - установить все необходимые зависимости проекта
4. `npm run build` - произвести сборку проекта
5. `npm run dev` - запустить проект на локальном сервере по адресу localhost:8080
